FLLTools Source Code Information

We maintain FLLTools as an open source software as open-source software under the Creative Commons BY-NC-SA license. You are free to use or modify the code as needed for use in your own software or personal applications.

We maintain three main versions of the software: FLLTools main version, Tournament edition, and an Embedded edition. The main version (http://flltools.flltutorials.com) is designed for home use. The tournament version (http://tournament.flltutorials.com) is designed for FLL tournnaments. The embedded edition is a reference implementation that can be used inside your own applications.

Home version: http://flltools.flltutorials.com
Tournament version: http://tournament.flltutorials.com
Embedded version: Core Values, Project, Robot Design, Scorer Translation software: http://flltools.flltutorials.com/translator

We also maintain (beta) utilities to generate new scoring and rubric tools (Rubric, Scorer,)
